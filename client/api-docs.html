API Documentation

Learn how to use the TK Media Downloader API

Overview

The TK Media Downloader API allows you to download media from supported platforms using a simple GET endpoint. It returns a clean JSON response with media links and metadata.

Endpoint

GET /down?url=<media-url>

Download media from a provided URL.

  • Parameter: url (required) - The URL of the media (e.g., YouTube, Instagram, etc.)
  • Supported Platforms: Facebook, Instagram, YouTube, TikTok, Twitter/X, Vimeo, Dailymotion, Pinterest

Example Request


curl "https://.onrender.com/down?url=https://www.youtube.com/watch?v=123456789"

Example Response


{
  "success": true,
  "error": null,
  "medias": [
    {
      "url": "https://example.com/hd.mp4",
      "quality": "HD"
    },
    {
      "url": "https://example.com/sd.mp4",
      "quality": "SD"
    }
  ],
  "metadata": {
    "title": "Sample Video",
    "duration": "00:03:25",
    "thumbnail": "https://example.com/thumb.jpg"
  }
}

Usage Notes

  • Ensure the URL is from a supported platform.
  • Use HTTPS for secure requests.
  • Respect rate limits (100 requests per 15 minutes).
  • Comply with platform copyright policies.
